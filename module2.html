Available AI Tools for Health Security Professionals

Several AI platforms are accessible to health security professionals. Each offers unique features and specializations, though all share common generative AI capabilities. Here are some of the most widely used platforms:

AI Platform Key Features Access Type Best For
Claude (Anthropic) Strong reasoning, document analysis, longer context windows Web interface, API, mobile app Complex reasoning tasks, document analysis, policy development
ChatGPT (OpenAI) Wide availability, web search capabilities, plugins Web interface, API, mobile app General-purpose tasks, quick research, prototyping
Gemini (Google) Web search integration, multimodal capabilities Web interface, API, mobile app Research-heavy tasks, image analysis
Specialized Health AI Tools Domain-specific capabilities, regulatory compliance Varies by provider Clinical applications, data analysis with regulatory requirements

For This Course:

We recommend using Claude or ChatGPT for the exercises in this course, as they offer free tiers with excellent capabilities for health security applications. While we'll demonstrate using Claude, the skills are transferable to other platforms.

Getting Started with Claude

Step 1: Creating an Account
  1. Visit claude.ai in your web browser
  2. Click "Sign Up" or "Create Account"
  3. Enter your email address
  4. Create a secure password
  5. Verify your email address (check your inbox for a verification link)
  6. Complete any additional verification steps if prompted
Step 2: Logging In
  1. Return to claude.ai
  2. Enter your email address and password
  3. Click "Log In" or "Sign In"
  4. You may need to complete a security verification step
Step 3: Navigating the Interface

Once logged in, you'll see:

  • Chat Area: The main space where you interact with the AI
  • Input Box: Where you type your prompts or questions
  • New Chat Button: To start a fresh conversation
  • History Panel: Shows your previous conversations
  • Settings: Access to account and conversation settings

Privacy and Security Considerations

When using AI tools in health security contexts, consider these important privacy and security factors:

  • Data Privacy: Information shared with public AI tools may be used for model training unless otherwise specified
  • Sensitive Information: Avoid sharing personally identifiable information (PII), protected health information (PHI), or confidential data
  • Tool Selection: Some organizations may have approved tools with stronger privacy guarantees
  • Organizational Policies: Follow your institution's guidelines for AI tool usage
Best Practices

✓ Use generalized examples rather than real case data

✓ Anonymize any data before sharing with AI tools

✓ Check if your organization has enterprise agreements with AI providers that offer enhanced privacy

✓ Be aware of data retention policies of the AI platform you're using

✓ Verify output with authoritative sources before using in critical applications
